Conversion from CSV to QIF

This page helps importing financial data into bookkeeping software. To be more precise, it converts CSV files (which you can obtain by downloading your financial data from your bank's website) to QIF files, which can be imported into bookkeeping software.

Currently it works for the following banks:

  • ABN AMRO (Nederland)
  • ING (Nederland)

The resulting QIF files are suitable for import in GnuCash and Microsoft Money, amongst others.

Update 7 October 2017: the converter has been entirely re-written from PHP to JavaScript. This means that the data files no longer need to be uploaded to the server. Instead, the conversion now will take place on your own computer. The old version is still available, but will be phased out soon.

Support for the following banks is not yet enabled in the new version, as I don't have example CSV files. To help me re-enable it, please send (the first couple of lines of) a CSV-file for these banks to me at qif@matthijsmelissen.nl

  • Rabobank (Nederland)
  • Spuerkees / BCEE (Banque et Caisse d'Epargne de l'Etat) (Luxembourg)
  • SEB Skandinaviska Enskilda Banken AB (Sweden)
